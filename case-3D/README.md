# List of files for 3D printing
This folder contains all the files for printing the box for the Timer
- 1x **case-top.stl** : The top part of the box should be printed in two colors. The bottom layers (0.5 mm) is black; the rest is gray PETG. 0.25 mm nozzle
- 1x **case-bottom.stl** : The bottom part of the box, printed in black PETG, 0.4 mm nozzle
- 1x **spacer-for-display.stl** : Spacer pad for the display. It is attached to the PCB.
- 1x **top-cable-holder.stl** : The top part of the cable holder. It is secured with two screws into the plastic.
- 4x **button-spacer.stl** : Fretboard pad. It slides onto the button, and the fretboard is then mounted onto it.

*.f3d files are Fusion360 files. You can edit individual parts before printing.
