# Time-switch-for-Pavol

Simple  timer switch for cars. Switches on the power supply to the heated mat for a set time. The time can be set and stored in one of four memory slots. It is controlled by four buttons and has an OLED display.

For details, see the Wiki (in Slovak) https://github.com/mateuskoOAMDG/Time-switch-for-Pavol/wiki
